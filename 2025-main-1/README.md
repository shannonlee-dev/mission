# 2025 Main 1

2025 메인 1차 과제 산출물을 보관하는 디렉토리입니다. 현재 구성은 Spaceship Titanic 데이터 분석 결과와 시각화 산출물 중심입니다.

## Structure

```text
2025-main-1/
├── README.md
└── spaceship-titanic/
    ├── analysis_transport.py
    └── transported_by_decade.png
```

## Contents

- `spaceship-titanic/analysis_transport.py`: Spaceship Titanic 데이터의 `Transported` 경향을 분석하는 Python 스크립트
- `spaceship-titanic/transported_by_decade.png`: 연령대 기준 운송 여부 시각화 결과

## Run

```bash
cd 2025-main-1/spaceship-titanic
python3 analysis_transport.py
```

## Notes

- 분석 코드는 과제별 하위 디렉토리에서 실행하는 것을 기준으로 합니다.
- 결과 이미지와 코드가 같은 과제 폴더 안에 있어 재현 산출물을 함께 확인할 수 있습니다.
