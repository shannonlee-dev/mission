# 2025 Selection 1

2025 선발 1차 과제 모음입니다. Python 기초 문법, 간단한 웹 프레임워크 조사, 계산기 구현, Git 개념 정리와 테스트 실습을 문제 단위로 분리했습니다.

## Structure

```text
2025-selection-1/
├── problem-1/
├── problem-2/
├── problem-3/
├── problem-4/
├── problem-5/
├── problem-6/
└── tools/
```

## Problem Index

| Path | Summary |
| --- | --- |
| `problem-1/` | Python 웹 프레임워크 조사 및 기초 풀이 |
| `problem-2/` | Flask 앱과 거듭제곱 계산기 |
| `problem-3/` | 웹 계산기 구현 및 업그레이드 |
| `problem-4/` | Git 디렉토리 구조, 코드 사용법, min/max 계산기 |
| `problem-5/` | 정렬 계산기와 staging/commit 개념 정리 |
| `problem-6/` | 우선순위 계산기, reset/revert 개념, 테스트 코드 |
| `tools/` | 보조 스크립트와 작업 메모 |

## Run

개별 문제 폴더로 이동한 뒤 Python 파일을 실행합니다.

```bash
cd 2025-selection-1/problem-6
python3 test_calc.py
```

Flask 기반 문제는 해당 폴더에서 앱 파일을 직접 실행합니다.

```bash
cd 2025-selection-1/problem-2
python3 app.py
```

## Notes

- 문제 폴더명은 `problem-N` 형식으로 통일했습니다.
- 학습 메모와 구현 코드를 같은 문제 폴더에 두어 평가자가 문제별 산출물을 한 번에 확인할 수 있게 했습니다.
